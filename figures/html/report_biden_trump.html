Twitter sentiment analysis

Analysis of topics and trends on Nov 4, 2020, the morning after the US election.

Trends of various n-gram frequencies during the data collection period are plotted here. For each trailing 5 minute time-period, we compute the frequencies of all possible n-grams. From this list, we plot the largest 20 n-grams.

While identifying the trends, the top-20 can be computed from any point in time. Since we are doing a retrospective analysis here, the top-20 list was derived from the final time.
